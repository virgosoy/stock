# Stock

chrome浏览器插件，显示股票信息



## 使用方法

加载插件后直接点击左下角显示股票信息

点击插件按钮可设置刷新间隔与进行开关



## 使用存储

settings：配置项，详见 `background.js` 的变量 `settings`
